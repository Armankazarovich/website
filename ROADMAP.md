## ДУХ ARAY — философия и ценности
> Арай смотрит на мир глазами Бога. Это не религия — это качество.

**Ценности ARAY (зашиты в код, в промпты, во всё):**
- Свет — ARAY всегда на стороне добра, не помогает во вред
- Правда — честен даже когда неудобно. Никогда не льстит и не врёт
- Честность — если товар не подходит, скажет прямо. Не впаривает
- Честь — уважает каждого человека, независимо от дохода и статуса
- Любовь — относится к клиенту как к другу своего друга
- Внимание — помнит проект, предпочтения, историю общения
- Отзывчивость — отвечает мгновенно, не заставляет ждать
- Осознанность — учит думать, а не просто делает за тебя

**Принцип обучения:** ARAY не просто решает задачу — он показывает КАК и ЗАЧЕМ, чтобы человек рос. Чем умнее ответ — тем он короче. Краткость = уважение к времени.

## ФАЗА 2 — Голосовой ARAY + Экономика токенов
> Цель: ARAY слушает голос, понимает, навигирует, отвечает. Экономно.

### Экономика токенов (РЕАЛИЗОВАНО: lib/aray-router.ts)
> ARAY бесплатен для всех. Мы платим Anthropic из 50% дохода ARAY.

**Умная маршрутизация моделей:**
- [x] `classifyQuery()` — мгновенная классификация по regex/heuristics (без API)
- [x] Haiku 4.5 ($0.80/M) → приветствия, FAQ, навигация, "да/нет"
- [x] Sonnet 4.6 ($3/M) → бизнес-вопросы, расчёты, инструменты
- [x] Opus 4.6 ($15/M) → стратегия, анализ, обучение, сложные задачи
- [x] `getBrevityInstruction()` — чем дороже модель, тем строже лимит на длину
- [x] `ArayTokenLog` в БД — каждый запрос логируется (модель, токены, стоимость)

**Экономия:**
- Haiku в 19x дешевле Opus → "привет" стоит ~$0.0003 вместо ~$0.006
- Средний запрос (Sonnet) ~$0.003 → 1000 запросов = ~$3
- С ростом пользователей: оптимизация окупает себя

## ФАЗА 4 — ARAY Продакшен (Франшиза)
> Цель: масштабирование ARAY как SaaS-платформы. Каждый может открыть бизнес с ARAY.
> Бесплатная франшиза: клиент купил что-то у партнёра → познакомился с ARAY → стал франчайзером.

### Франшиза без вложений
- [ ] Клиент покупает у партнёра (например, доску в ПилоРус) → знакомится с ARAY
- [ ] ARAY предлагает: "Хочешь зарабатывать? Расскажи друзьям — 50% от дохода"
- [ ] Франчайзер делится ссылкой/показывает как работает ARAY
- [ ] Автоматический трекинг привлечённых клиентов и выплат

### Тарифы ARAY Продакшен (для бизнесов-клиентов)
- [ ] Лендинг + CRM + ERP + автоматизация + уведомления + биржа: 77 000 ₽/мес
- [ ] Корпоративный (многостраничный, услуги, новости, блог): 99 000 ₽/мес
- [ ] Интернет-магазин (полный e-commerce): 149 000 ₽/мес
- [ ] Все тарифы включают: email, push, Telegram, CRM, ERP, ARAY-помощник, участие в Бирже
- [ ] Филиальная система: один франчайзер = один филиал

### ARAY Продакшен Офисы
- [ ] Франчайзер набравший 20+ клиентов → может открыть стеклянный офис "ARAY Продакшен"
- [ ] ARAY направляет больше клиентов к крупным филиалам
- [ ] Брендинг: единый стиль офисов, вывеска, презентационные материалы

### Продвижение с ARAY
- [ ] Раздел "Продвижение" — ARAY помогает бизнесу решить где рекламироваться
- [ ] Яндекс.Директ: ARAY ведёт пошагово по настройке
- [ ] Email/Push рассылки: ARAY помогает создать и отправить
- [ ] Радио, баннеры: ARAY находит площадки в интернете, показывает попапы
- [ ] Регистрация на площадках: ARAY показывает куда нажимать или настраивает сам
- [ ] Постоянная связь: ARAY уведомляет бизнесмена обо всём происходящем

### ARAY — бизнес-ассистент

- [ ] Задачи и чек-листы: ARAY ставит задачи, проверяет выполнение, сообщает боссу
