## ФАЗА 1 — КРИТИЧНО ДЛЯ ЗАПУСКА (1-2 дня)

### 1.1 Голос Арая — TTS что реально работает
**Проблема:** ElevenLabs заблокирован из России по IP. Браузерный TTS не находит русский голос на многих устройствах.

**Решение:**
- [ ] **Yandex SpeechKit TTS** — работает из России, русский голос нативный, бесплатный тариф 500K символов/мес
  - API: `POST https://tts.api.cloud.yandex.net/speech/v1/tts:synthesize`
  - Голос: `filipp` (мужской, тёплый) или `alena` (женский)
  - Формат: OGG opus → легко играть в браузере
- [ ] Fallback цепочка: Yandex SpeechKit → ElevenLabs → Browser SpeechSynthesis
- [ ] Кнопка "Озвучить" на каждом сообщении Арая (уже есть, но не работает из-за TTS)
- [ ] Авто-озвучка опционально (уже есть toggle в admin)

**Файлы:** `app/api/ai/tts/route.ts`, env: `YANDEX_CLOUD_API_KEY`, `YANDEX_CLOUD_FOLDER_ID`

### 1.2 Голосовой ввод (STT) — улучшение
**Проблема:** Browser SpeechRecognition работает нестабильно, особенно на iOS.

**Решение:**
- [ ] **Yandex SpeechKit STT** как серверный fallback — записать аудио через MediaRecorder → отправить на сервер → распознать
  - API: `POST https://stt.api.cloud.yandex.net/speech/v1/stt:recognize`
  - Формат: OGG opus
- [ ] Фронт: MediaRecorder → запись → отправка на `/api/ai/stt`
- [ ] Fallback: если браузерный SpeechRecognition работает — использовать его (быстрее)
- [ ] Визуал: анимация волны при записи (уже есть пульс, но улучшить)

**Файлы:** NEW `app/api/ai/stt/route.ts`, обновить виджеты

### 1.3 Загрузка файлов в чат (фото, документы)
**Проблема:** Нельзя отправить фото товара, прайс-лист, документ.

**Решение:**
- [ ] Кнопка 📎 скрепка рядом с микрофоном в обоих виджетах
- [ ] Принимает: изображения (jpg/png/webp), PDF, Excel (xlsx), текстовые файлы
- [ ] Upload → сервер сохраняет в `/tmp/aray-uploads/` или S3
- [ ] Для изображений: отправляем в Claude API как `image` content block (Claude Vision)
- [ ] Для PDF: извлекаем текст → отправляем как текст
- [ ] Для Excel: парсим → отправляем как таблицу
- [ ] Превью загруженного файла в пузыре сообщения

**Юзкейсы:**
- Фото товара → "Добавь этот товар в каталог" → Claude видит фото, описывает, создаёт товар
- Фото прайс-листа → "Загрузи эти цены" → Claude распознаёт таблицу, импортирует
- Скриншот ошибки → "Что это?" → Claude объясняет

**Файлы:** 
- NEW `app/api/ai/upload/route.ts`
- Обновить `app/api/ai/chat/route.ts` — поддержка image content blocks
- Обновить оба виджета — UI загрузки

### 1.4 Попап-браузер в админке
**Проблема:** В админке нет ArayBrowser — `show_url` просто открывает новую вкладку.

**Решение:**
- [ ] Импортировать `ArayBrowser` компонент в `admin-aray.tsx`
- [ ] При `show_url` и `show_page` → открывать попап поверх админки
- [ ] Для внутренних URL (`/admin/...`, `/catalog/...`) → iframe попап
- [ ] Для внешних URL → iframe попап (если X-Frame-Options позволяет) или ссылка

### 1.5 Попап для store-виджета — улучшение
**Проблема:** Сейчас `navigate_page` делает полный редирект вместо попапа.

**Решение:**
- [ ] Новый инструмент `show_page` (уже добавлен в код)
- [ ] Новая команда `__ARAY_POPUP:{json}__` вместо хрупких regex
- [ ] По умолчанию — всё в попапе, полный переход только по явному запросу
- [ ] Обновить промпты чтобы ARAY знал когда popup, когда redirect

### 1.6 Память чата — не забывать контекст
**Проблема:** При перезагрузке страницы чат обнуляется.

**Решение:**
- [ ] Сохранять историю в `localStorage` (последние 20 сообщений)
- [ ] При открытии чата — восстанавливать историю
- [ ] Кнопка "Новый чат" — очищает историю
- [ ] Серверная память уже есть (`aray-memory.ts`) — используем для долгосрочного контекста

## ТЕХНИЧЕСКИЕ ЗАМЕТКИ

### Yandex SpeechKit — регистрация
1. console.cloud.yandex.ru → создать каталог
2. Сервисный аккаунт с ролью `ai.speechkit-tts.user`
3. IAM-токен или API-ключ
4. env: `YANDEX_CLOUD_API_KEY`, `YANDEX_CLOUD_FOLDER_ID`

### Claude Vision — загрузка изображений
```typescript
// В messages передаём image как base64
{
  role: "user",
  content: [
    { type: "image", source: { type: "base64", media_type: "image/jpeg", data: base64str } },
    { type: "text", text: "Что на этом фото? Создай товар по этому изображению." }
  ]
}
```

### Структура загрузки файлов
```
POST /api/ai/upload → { fileId, type, preview }
POST /api/ai/chat → { messages, context, files: [{ fileId, type }] }
```
